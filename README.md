# South Peak Construction

Professional construction and renovation landing page for South Peak Construction, serving the Gauteng region. This application features a modern design, a project gallery, and an AI-powered quote assistant.

## Features

- **Hero Section**: Engaging landing area with strong branding.
- **Services Showcase**: Detailed list of construction and renovation services offered.
- **Project Gallery**: Visual portfolio of past projects.
- **AI Vision**: AI-powered features utilizing Google's Gemini API for vision and analysis.
- **Quote Form**: Interactive form for potential clients to request quotes.
- **Responsive Design**: Fully responsive layout optimized for all devices.

## Tech Stack

- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **AI Integration**: Google GenAI SDK (Gemini)
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   - Create a `.env` file in the root directory.
   - Add your Gemini API key:
     ```env
     GEMINI_API_KEY=your_api_key_here
     ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Runs the linter to check for code quality issues.
